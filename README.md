fresh
=====

Fresh Programming Language

Features:

- Supports Booleans
- true and false
- not Get Not of given boolean

- Supports Integers
- + Addition
- - Subtraction
- * Multiplication
- = Equality Checking
- > Greater than
- < Less than

- Supports Conditional Branching
- if branch based on condition
